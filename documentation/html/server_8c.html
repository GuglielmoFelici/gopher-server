int installDefaultSigHandlers ( )

Installs the default signal handlers for the main process. On Windows, CTRL_C interrupts the server and CTRL_BREAK requests a configuration update. On Linux, SIGINT interrupts the server and SIGHUP requests a configuration update.

Returns
SERVER_SUCCESS or SERVER_FAILURE.
