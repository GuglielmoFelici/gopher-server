Executes the gopher protocol. Receives a selector and performs validateInput() and normalizePath(). If the selector points to a directory, calls sendDir(). Else, a file_mapping_t is created using getFileMap() and is sent to the client using sendFile().

Returns
GOPHER_SUCCESS or GOPHER_FAILURE.
