Executes the gopher protocol for the client socket sock.

Sock will be closed when the function returns. The function can spawn a helper thread to send a file, so the transmission will be aborted if the process termiates.

Parameters
sockthe connected socket of the client
portthe port where the resources can be found
pLoggeran optional pointer to a logger_t struct used for logging the transfer.
Returns
GOPHER_SUCCESS or GOPHER_FAILURE.
