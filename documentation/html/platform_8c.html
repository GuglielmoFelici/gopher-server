◆ iterateDir()

int iterateDir ( cstring_t  path,
_dir *  dir,
string_t  name,
size_t  nameSize 
)

Reads the next files of a directory and copies its relative path in name. If dir is NULL, opens the directory path and initializes dir for further calls. If dir is non-NULL, path is ignored (thus can be NULL), and the scan continues from the previouse iterateDir() call. NOTE: dir must be closed using closeDir() when done.

Parameters
pathThe path to a directory to open. Ignored if dir is non-NULL.
dirAn object representing a directory. Used after the first call to iterateDir() for subsequent calls.
nameA buffer where the file relative path is to be copied.
nameSizeThe size of the buffer name.
Returns
PLATFORM_SUCCESS or PLATFORM_FAILURE. PLATFORM_FAILURE can be or-ed with PLATFORM_NOT_FOUND or PLATFORM_END_OF_DIR.
